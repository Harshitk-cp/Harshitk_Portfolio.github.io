Live Room Chat Application

I started this project recently to test my Node.js skill I had learned in past few months.

This app works on both backend and front end. The user inputs their name to enter the chatting room and the server connects them to a localhost server.

Once inside the room every user that is already inside the server can communicate with one other via texting or sharing images. Their is no database involved in this so, every communication happens through live servers. The images is shared between devices with base64 encoding.

I only recently started this application and will add new features to it in coming future so stay connected to check them out.

Check out project Repository
