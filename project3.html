Apperal Search & Try on application

This project uses some advance concepts like machine learning and image extraction which I'm still learning about so I made this application to test my skills.

This app inputs clothes of user's choice and shows results of similar clothes available online. once user selects the apperal of his/her choice user can user our virtual try-on feature to see how the said apperal will look on then.

This is a big project and took a lot of time to work on and is still under progress as there are multiple issues regarding image search and image extraction.

I started working on this project few months ago and have not gotten time to finish it but i'll be sure to complete it as I gain more knowledge regarding such subjects.
