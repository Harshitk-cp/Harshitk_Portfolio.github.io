Online Coffee Ordering Application

Instead of using Kotlin I wanted to build a project in Java so i made this coffee shopping app. It takes details of coffee from a firebase Database as a json class of objects and shows it in a Recycler View.

By clicking on any of the item we can add it to cart and in Cart Activity we can increase or decrease quantity. Then finally inside Preview Activity user can place their order.

I've used multiple functionality in this project such as recycler view, navigation view, user Authentication, Databses, etc.

This project is close for now as I don't have anything to add to this project.
